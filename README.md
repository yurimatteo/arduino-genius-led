# arduino-genius-led
Meu primeiro projeto arduino de verdade onde demonstro como fazer um jogo Genius

## Pré-requisitos

* Arduino UNO
* 4 LEDs nas cores azul, vermelho, verde e amarelo
* 3 resistores de 330 Ohms
* 13 fios com conectores machoXmacho
